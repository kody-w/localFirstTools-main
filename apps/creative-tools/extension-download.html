üîß Troubleshooting

‚ùå Error: "This extension may have been corrupted"

Solution:

  1. Remove the extension
  2. Re-extract the ZIP file to a new location
  3. Make sure you're not extracting to a network drive or cloud folder
  4. Try loading the extension again
‚ùå Extension disappeared after browser restart

Solution: You may have moved or deleted the extension folder. Chrome needs continuous access to the unpacked files.

  1. Extract the extension to a permanent location (like Documents folder)
  2. Do NOT delete the folder after installation
  3. If you need to move it, remove the extension first, then reinstall from new location
‚ùå Can't find "Load unpacked" button

Solution: Make sure Developer mode is enabled (toggle in top-right corner of chrome://extensions/)

‚ùì Apps not loading or showing 404 errors

Solution: The extension folder structure may be incorrect.

  1. Make sure you selected the folder containing manifest.json (not a parent folder)
  2. Verify the apps/ and data/ folders exist inside
  3. Try reloading the extension (click refresh icon on extension card)
‚ùì How do I update the extension?

Solution:

  1. Download the latest version
  2. Remove the old extension from Chrome
  3. Extract new version to same location (or new location)
  4. Load the new version using "Load unpacked"
